<strong>CIFS Documents Provider</strong> is an Android app to provide access to shared online storage.

<strong>Features</strong>

<ul>
<li>Provide other apps with access to shared network storage via the Storage Access Framework (SAF).</li>
<li>Provides access to files and directories.</li>
<li>Supports SMB (Samba, Common Internet File System (CIFS), Windows Network Shared Folder), FTP, FTPS and SFTP.</li>
<li>Share and transfer files on online storage.</li>
<li>Multiple connection settings can be stored.</li>
<li>Supports dark mode.</li>
<li>Can be treated as local storage (configuration required).</li>
<li>Notifications can be displayed to prevent task kills (configuration required).</li>
</ul>

<strong>Objective</strong>

<ul>
<li>Import and export of files created by the app.</li>
<li>Manage files and directories with the Storage Manager app.</li>
<li>Play music, videos, etc. with the media player app.</li>
<li>Direct saving of photos taken with the camera app.</li>
</ul>

<strong>Note</strong>

<ul>
<li>No file management function in this app.</li>
<li>To use this app, your apps must support SAF (Storage Access Framework).</li>
<li>Apps that assume local storage may not work properly.</li>
<li>Apps may crash when specified as a storage destination for streaming audio or video data.</li>
</ul>
