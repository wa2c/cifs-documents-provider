<strong>Провайдер документов для CIFS/SMB</strong> это приложение предоставляющее доступ к общему онлайн хранилищу.

<strong><font color="#339933">Возможности</font></strong>

* Позволяет другим приложениям работать с общим сетевым хранилищем через Storage Access Framework (SAF).
* Предоставляет доступ к файлам и папкам.
* Поддерживает SMB (Samba, Common Internet File System (CIFS), Windows Network Shared Folder), FTP, FTPS и SFTP.
* Поделиться и передавать файлы на сетевое хранилище.
* Может использовать несколько настроек соединений.
* Поддерживается тёмный режим.
* Может представляться локальным хранилищем (требует настройки).
* Может показывать уведомления, чтобы избежать остановки задач (требует настройки).

<strong><font color="#339933">Использование</font></strong>

* Импортировать и экспортировать файлы созданные приложением.
* Работать с файлами и папками из приложения Менеджер Хранилища.
* Проигрывать музыку, видео, и т.д. в медиа плеере.
* Сохранение напрямую снятых фотографий из приложения камеры.

<strong><font color="#339933">Примечания</font></strong>

* Нет функциональности управления файлами в самом приложении.
* Для использования этого провайдера, ваши приложения должны поддерживать SAF (Storage Access Framework).
* Приложения могут предполагать что локальное хранилище может не работать правильно.
* Приложения могут отпадать когда хранилище указано как назначение для аудио-потока или видео данных.
