<strong>Провайдер CIFS документов</strong> это Android приложение предоставляющее доступ к общему онлайн хранилищу.


<strong>Возможности</strong>

<ul>
<li>Позволяет другим приложениям работать с общем сетевым хранилищем через Storage Access Framework (SAF).</li>
<li>Предоставляет доступ к файлам и папкам.</li>
<li>Поддерживает SMB (Samba, Common Internet File System (CIFS), Windows Network Shared Folder), FTP, FTPS и SFTP.</li>
<li>Поделиться и передавать файлы на сетевое хранилище.</li>
<li>Может использовать несколько настроек соединений.</li>
<li>Поддерживается тёмный режим.</li>
<li>Может представляться локальным хранилищем (требует настройки).</li>
<li>Может показывать уведомления чтобы избежать остановки задач (требует настройки).</li>
</ul>

<strong>Использование</strong>

<ul>
<li>Импортировать и экспортировать файлы созданные приложением.</li>
<li>Работать с файлами и папками из приложения Менеджер Хранилища.</li>
<li>Проигрывать музыку, видео, и тд. в медиа плеере.</li>
<li>Сохранение напрямую снятых фотографий из приложения камеры.</li>
</ul>

<strong>Примечания</strong>

<ul>
<li>Нет функциональности управления файлами в самом приложении.</li>
<li>Чтобы использовать это приложение, ваши приложения должны поддерживать SAF (Storage Access Framework).</li>
<li>Приложения могут предполагать что локальное хранилище может не работать правильно.</li>
<li>Приложения могут отпадать когда хранилище указано как назначение для потока аудио или видео данных.</li>
</ul>
